# JLPT N4 学習アプリ 問題作成指示書

## アプリの概要

技人国ビザを持つフィリピン人向けのJLPT N4合格を目指す日本語学習アプリです。
- モバイルファースト・シングルHTMLファイル（`jlpt-n4.html`）
- 試験モード：全問回答 → 結果画面で正解・不正解をまとめて表示
- 解説は**日本語＋タガログ語**の両方で記載
- 学習記録はGoogleスプレッドシートに自動保存

## 問題セクションの種類（1レッスンにつき5種類）

### 問題１：穴埋め問題
```
instruction: "（　）に なにを いれますか。いちばん いい ものを ひとつ えらんで ください。"
形式: 文の中の（　）に入る語句を4択から選ぶ
```

### 問題２：言い換え問題
```
instruction: "＿＿の ぶんと だいたい おなじ いみの ぶんが あります。いちばん いい ものを ひとつ えらんで ください。"
形式: 下線部（<u>テキスト</u>）と同じ意味の文を4択から選ぶ
```

### 問題３：語彙の使い方
```
instruction: "つぎの ことばの つかいかたで いちばん いい ものを ひとつ えらんで ください。"
形式: 「単語」の正しい使い方の文を4択から選ぶ
```

### 問題４：漢字の読み方（ひらがな）
```
instruction: "ひらがなで どう かきますか。ひとつ えらんで ください。"
形式: 下線部の漢字（<u>漢字</u>）の読み方を4択から選ぶ
```

### 問題５：漢字の書き方
```
instruction: "どう かきますか。ひとつ えらんで ください。"
形式: 下線部のひらがな（<u>ひらがな</u>）の漢字表記を4択から選ぶ
```

## データ形式（JavaScriptオブジェクト）

```javascript
{
  id: 5,                          // レッスン番号（整数）
  title: "タイトル",              // 日本語タイトル
  subtitle: "English subtitle",   // 英語サブタイトル
  icon: "🎯",                     // 絵文字アイコン
  questions: [
    {
      section: "問題１",          // セクション名
      instruction: "（　）に...", // 問題の指示文
      q: "問題文（　）。",        // 問題文 ※下線はHTMLタグ<u>テキスト</u>で表記
      c: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],  // 4つの選択肢
      a: 0,                        // 正解のインデックス（0〜3）
      exp: {
        ja: "日本語の解説文。なぜこれが正解か、他の選択肢がなぜ違うかを説明。",
        tl: "Tagalog na paliwanag. Ipaliwanag kung bakit tama ang sagot at bakit mali ang iba."
      }
    },
    // ... 合計12問（問題１×4問、問題２×2問、問題３×2問、問題４×2問、問題５×2問）
  ]
}
```

## 問題数の目安

1レッスン = **12問**
- 問題１（穴埋め）：4問
- 問題２（言い換え）：2問
- 問題３（語彙の使い方）：2問
- 問題４（漢字→ひらがな）：2問
- 問題５（ひらがな→漢字）：2問

## 解説の書き方ルール

### 日本語解説（exp.ja）
1. 正解の語句の意味を説明（漢字・英語の意味も添える）
2. 正解の文脈・使い方を説明
3. 不正解の選択肢がなぜ違うかを番号付きで説明
4. 難易度：N4レベル学習者向けのわかりやすい言葉で

**例：**
```
「きょうみ（興味）」は「関心・好きな気持ち」という意味です。「〜に興味がある」は
「〜が好きで、もっと知りたい」という表現です。②しょうせつ＝小説（novel）、
③せいさん＝生産（production）、④ちから＝力（power）は文に合いません。
```

### タガログ語解説（exp.tl）
1. 日本語の単語の意味をタガログ語で説明
2. 正解の理由をフィリピン人に伝わる表現で
3. 間違いやすいポイントを指摘

**例：**
```
Ang 'kyoumi (興味)' ay nangangahulugang interes. 'Eiga ni kyoumi ga aru' = 
interesado sa pelikula. Ang ibang mga salita ay hindi angkop sa pangungusap.
```

## ファイルへの追加方法

`jlpt-n4.html` 内の `const LESSONS = [` 配列に追加します。

### 現在のLESSONS配列の末尾（id:4の後）に追加する：

```javascript
  {
    id:5, title:"〇〇〇〇", subtitle:"English subtitle", icon:"🎯",
    locked: false,  // ← falseにしないとロックがかかる
    questions:[
      // ここに12問分のオブジェクトを記載
    ]
  },
```

### ロック設定について
- `locked: true` または省略 → ロックがかかり学習不可
- `locked: false` → 学習可能

現在id:5〜8は `locked: true` になっています。問題を追加するときは `locked: false` に変更してください。

## 注意事項

- 問題文中の下線部は `<u>テキスト</u>` で表記すること
- 選択肢（c配列）は必ず4つ
- 正解インデックス（a）は0から始まる（1番目=0、2番目=1、3番目=2、4番目=3）
- 文字列内にダブルクォーテーション `"` を使うときはシングルクォーテーション `'` または全角「」に変換すること（JavaScriptエラーの原因になる）
